原生微信小程序安装Eslint

1. VsCode安装Eslint插件

2. cd到我们的项目，安装ESlint

eslintrc.jscnpm install eslint --save-dev

3. 创建ESlint配置文件.eslintrc.js

npx eslint --init
#或者
eslint --init

4. eslintrc.js文件生成之后，我们就可以使用基本的命令规范我们的代码了

eslint --fix pages/index/index.js

当然，如果你使用了ES6的语法，上述命令执行后会报如下错误

Parsing error: Unexpected token = ***

原因是开发环境与ESLint当前的解析功能不兼容，因此我们需要执行下一个步骤；

cnpm install babel-eslint --save-dev

安装完成后，在.eslintrc中添加配置："parser": "babel-eslint" 即可；

6. 修改 package.json

 "scripts": {
    "lint": "npx eslint --fix **/*.js"
  },

7. 创建完package.json文件，就可以执行如下命令对指定的文件进行eslint管理：

npm run lint
