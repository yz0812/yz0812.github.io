maven

1.配置本地路径

修改maven下的conf/settings.xml 文件

<settings xmlns="http://maven.apache.org/SETTINGS/1.0.0"
          xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
          xsi:schemaLocation="http://maven.apache.org/SETTINGS/1.0.0 http://maven.apache.org/xsd/settings-1.0.0.xsd">
		   <localRepository>D:\cache\mavenRepository</localRepository>

2.设置阿里云镜像

找到mirrors,添加


 <mirror>
      <id>alimaven</id>
      <name>aliyun maven</name>
      <url>http://maven.aliyun.com/nexus/content/groups/public/</url>
      <mirrorOf>central</mirrorOf>        
    </mirror>

MySQL

1.下载

下载 解压版

2.解压放到目录设置环境变量

在path 下新增 D:\Program Files\mysql-8.0.18-winx64\bin

3.在mysql的根目录下新建 my.ini文件

[mysqld]
port = 3306
basedir=D:/Program Files/mysql-8.0.18-winx64
datadir=D:/Program Files/mysql-8.0.18-winx64/data 
max_connections=200
character-set-server=utf8
default-storage-engine=INNODB
sql_mode=NO_ENGINE_SUBSTITUTION,STRICT_TRANS_TABLES
[mysql]
default-character-set=utf8

注意修改basedir和datadir,并且斜杠必须为/不是\ ,不然会报错 ,正确:D:/Program Files/mysql-8.0.18-winx64,错误:D:\Program Files\mysql-8.0.18-winx64

4.cmd安装

以管理员身份打开cmd命令窗口，将目录切换到MySQL的安装目录的bin目录下

安装

mysqld -install

MySQL的初始化,并且会生成data文件夹

mysqld --initialize-insecure --user=mysql

开启MySQL服务

net start mysql

修改密码

mysqladmin -u root -p password 新密码
Enter password: 旧密码
install
