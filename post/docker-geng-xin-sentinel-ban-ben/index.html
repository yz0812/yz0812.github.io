Docker更新Sentinel版本

sentinel springcloud
2021-05-17

本机环境 macOS Big Sur 11.0.1

docker v20.10.5

1.将sentinel打包成jar

在配置文件中使用127.0.0.1:8848的地址无效,需要使用本机内网IP

2.进入sentinel容器

#查看container_id
docker container ls  
#进入容器
sudo docker exec -it -u root 52fe0732a674 /bin/bash 
#查看地址
pwd

3.上传文件到容器

#下载到本地 docker cp container_id:容器地址 本地地址
docker cp 52fe0732a674:/bladex/sentinel/app.jar /Users
#上传到容器 docker cp 本地地址 container_id:容器地址
docker cp /Users/workPackage/idea/Sentinel/sentinel-dashboard/target/sentinel-dashboard.jar 52fe0732a674:/bladex/sentinel/

4.重命名

#查看是否上传成功
bash-4.4# ls
app.jar                 sentinel-dashboard.jar
#删除原有的 app.jar
bash-4.4# rm app.jar
#重命名
bash-4.4# mv sentinel-dashboard.jar app.jar

5.重新启动Docker即可
