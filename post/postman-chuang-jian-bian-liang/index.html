Postman创建变量

install
2020-05-11

前言在学习security中使用postman获取access_token,每次token都不一样每次都要复制太麻烦了.后找到解决方法

1.创建全局环境

点击设置按钮

点击Globals

VARIABLE 变量名
INITIAL VALUE 初始值
CURRENT VALUE 当前值

2.创建局部环境

点击设置按钮

点击Add

Add Environment 环境名称
VARIABLE 变量名
INITIAL VALUE 初始值
CURRENT VALUE 当前值

到目前我们添加了 loca全局变量,dev局部变量 参数为access_token,refresh_token,接下来告诉大家如何使用

3.查看变量


第一个 No Environment是选择局部变量
第二个是查看全部变量
第三个是编辑

4.赋值

pm.test("Status code is 200", function () {
    pm.response.to.have.status(200);
    let json = JSON.parse(responseBody);//声明一个变量json，将JSON返回Body里面的值解析并给变量json
    tests["a"] = json.expires_in ===7200;//设置一个断言
    pm.environment.set("refresh_token", json.refresh_token);
    pm.environment.set("access_token", json.access_token);
});
